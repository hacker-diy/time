# Time Trick Lock Screen

PWA-like single page that imitates an iOS-style lock screen for a magic trick.

## Features

- Uses real device time
- Weather widget via Open-Meteo API + geolocation
- Secret settings menu (5x fast tap on camera icon)
- Trick parameters:
  - Time offset in minutes (default +5)
  - Delay before rewind in seconds (default 5)
  - Rewind speed multiplier (default 100)
- Black screen cover before the trick
- Custom wallpaper upload
- Reset button returns default parameters and design

To use on GitHub Pages, just push this folder as a repo and enable Pages with root as the source.
